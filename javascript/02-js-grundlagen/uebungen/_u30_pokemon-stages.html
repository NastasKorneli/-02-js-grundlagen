Übung 30: Pokémon — Evolution

Kennen Sie Pokémon? Allen Pokémon-Spielen ist gemein, dass es Pokémon (Phantasiewesen) gibt, die über verschiedene Evolutionsstufen verfügen. Ein Pokémon kann sich bis zu zweimal weiterentwickeln. D.h. es gibt maximal drei Stufen: seine Anfangsstufe und bis zu zwei Folgestufen. Jede dieser Stufen trägt einen neuen Pokémon-Namen — keiner dieser Namen kann doppelt vorkommen.

  1. Schreiben Sie eine Funktion stagesFor, der Sie eine Pokémon-Evolutionsstufe übergeben und die Ihnen alle dazugehörigen Evolutionsstufen als Array zurückliefert.

  2. Entwickeln Sie die Funktionen stagesAfter und stagesBefore, die Ihnen nur die Stufen vor bzw. nach der angefragten Stufe zurückliefern.
